## О программе

*Создание односвязанного списка:*

1. *С реализацией функций: добавление, удаление элементов списка, печать списка на экране.*

## Как запустить (linux)

*Для запуска программы небходимо:*

1. `g++ -Wall one_spisk.cpp -o one_spisk`
